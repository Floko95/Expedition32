Readme - HDRP Glass Shaders
___________________________


Support Unity versions
2019.4 LTS
2020.3 LTS
2022.3 LTS (by default)

By default, the package is set up to be compatible for Unity 2022.3.x. If you are using Unity 2020.3.x or 2021.3.x, 
unpack the HDRP-Glass Shaders_2020.3.x.unitypackage.


Documentation link : https://docs.google.com/document/d/1kvqsWU2jcxF-xhudj-5Yvjy17jKQ70ZTNWqqFMI0Glw/edit?usp=sharing
